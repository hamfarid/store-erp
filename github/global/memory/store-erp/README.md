# AI Helper Memory System

This directory contains the AI Senior Technical Lead's memory system.

## Purpose
Enable context retention, decision tracking, and learning across sessions.

## Structure
- `conversations/` - Conversation history
- `knowledge/` - Knowledge base
- `preferences/` - User preferences
- `state/` - Current state
- `checkpoints/` - State checkpoints
- `decisions/` - Decision logs
- `summaries/` - Conversation summaries

This is the AI's HELPER TOOL, not part of user's project!
