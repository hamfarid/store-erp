# MCP (Model Context Protocol) System

This directory contains MCP server configurations.

## Purpose
Provide access to external tools and services.

## Available Servers
- **cloudflare** - Cloudflare Workers, D1, R2, KV
- **playwright** - Browser automation
- **sentry** - Error monitoring
- **github** - GitHub integration

## Configuration
Edit `config/mcp_config.json` to enable/configure servers.
