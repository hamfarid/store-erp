# دليل المستخدم لنظام الذكاء الاصطناعي الزراعي المتكامل

## مقدمة

نظام الذكاء الاصطناعي الزراعي المتكامل هو منصة متطورة تجمع بين رؤية حاسوبية ومعالجة لغة طبيعية وتركيب معرفي لمحاكاة خبرة المهندسين الزراعيين. يوفر النظام مجموعة شاملة من الأدوات لتحليل الأمراض النباتية، نقص العناصر الغذائية، التهجين النباتي، وتوصيات العلاج.

تم تطوير هذا النظام باستخدام Manus، وهو يتميز بالقدرة على التعلم المستمر وتحديث قواعد البيانات تلقائيًا من مصادر موثوقة على الإنترنت.

## المميزات الرئيسية

### 1. الكشف عن الأمراض النباتية
- تحليل الصور للكشف عن الأمراض الفيروسية والبكتيرية والفطرية
- تحديد الأعراض وشدة الإصابة
- تقديم تشخيص دقيق مع درجة ثقة

### 2. تحليل نقص العناصر الغذائية
- تحديد نقص العناصر الغذائية الرئيسية والثانوية
- تحليل أعراض نقص التغذية من خلال الصور
- تقديم تقارير مفصلة عن حالة النبات

### 3. التهجين وتزاوج النباتات
- التنبؤ بنتائج التهجين بناءً على خصائص السلالات الأبوية
- اقتراح أزواج تهجين لتحقيق صفات مستهدفة
- توثيق نتائج التهجين من مصادر موثوقة

### 4. توصيات العلاج
- تقديم توصيات دقيقة للعلاج بناءً على التشخيص
- حساب الجرعات المناسبة وتوقيت التطبيق
- تقديم بدائل علاجية متعددة

### 5. التعلم المستمر وتحديث النماذج
- تحسين النماذج تلقائيًا بناءً على التغذية الراجعة
- تحديث قاعدة المعرفة باستمرار
- تكيف النظام مع الظروف المتغيرة

### 6. جمع البيانات وتوثيق المصادر
- جمع المعلومات والصور من مصادر موثوقة على الإنترنت
- تقييم موثوقية المصادر وتوثيقها
- بناء قاعدة بيانات شاملة ومحدثة

## متطلبات النظام

- نظام تشغيل: Linux/Windows/macOS
- Python 3.8 أو أحدث
- وحدة معالجة رسومات (GPU) موصى بها للأداء الأمثل
- اتصال إنترنت للوصول إلى المصادر وتحديث قاعدة البيانات

## التثبيت

1. استنساخ المستودع:
```bash
git clone https://github.com/your-organization/agricultural-ai-system.git
cd agricultural-ai-system
```

2. إنشاء بيئة افتراضية:
```bash
python -m venv venv
source venv/bin/activate  # على Linux/macOS
venv\Scripts\activate  # على Windows
```

3. تثبيت المتطلبات:
```bash
pip install -r requirements.txt
```

4. تهيئة قاعدة البيانات:
```bash
python src/database/init_database.py
```

5. تنزيل النماذج المدربة مسبقًا:
```bash
python src/models/download_models.py
```

## كيفية الاستخدام

### واجهة المستخدم الرسومية

1. تشغيل الواجهة الرسومية:
```bash
python src/web_interface/app.py
```

2. فتح المتصفح والانتقال إلى `http://localhost:5000`

### تحليل صورة نباتية

1. تحميل صورة للنبات المصاب
2. اختيار نوع النبات من القائمة المنسدلة
3. النقر على زر "تحليل"
4. استعراض النتائج التي تتضمن:
   - الأمراض المكتشفة
   - نقص العناصر الغذائية
   - توصيات العلاج

### البحث عن معلومات نباتية

1. إدخال استعلام البحث في حقل البحث
2. النقر على زر "بحث"
3. استعراض النتائج من قاعدة البيانات المحلية والمصادر الموثوقة على الإنترنت

### التنبؤ بنتائج التهجين

1. اختيار السلالة الأبوية الأولى
2. اختيار السلالة الأبوية الثانية
3. النقر على زر "تنبؤ"
4. استعراض النتائج المتوقعة للتهجين والمراجع الخارجية

## الميزات المتقدمة

### إضافة ملاحظات للتعلم المستمر

يمكن للمستخدمين المساهمة في تحسين النظام من خلال:

1. تصحيح التشخيصات الخاطئة
2. تأكيد التشخيصات الصحيحة
3. إضافة معلومات جديدة إلى قاعدة البيانات

### تخصيص النظام

يمكن تخصيص النظام من خلال تعديل ملف التكوين `config/default.yaml`:

- تغيير مسارات الحفظ
- ضبط معايير التحقق من المصادر
- تعديل إعدادات التعلم المستمر
- تخصيص معايير البحث

## استكشاف الأخطاء وإصلاحها

### مشاكل تحليل الصور

- تأكد من أن الصورة واضحة وبدقة كافية
- تأكد من أن الصورة تظهر الأعراض بوضوح
- جرب التقاط الصورة في إضاءة أفضل

### مشاكل الاتصال بالإنترنت

- تحقق من اتصال الإنترنت
- تأكد من أن جدار الحماية لا يمنع الوصول إلى المصادر
- جرب استخدام وضع عدم الاتصال إذا كانت قاعدة البيانات المحلية محدثة

### مشاكل أخرى

- راجع ملفات السجل في مجلد `logs/`
- أعد تشغيل النظام
- تحقق من وجود تحديثات للنظام
