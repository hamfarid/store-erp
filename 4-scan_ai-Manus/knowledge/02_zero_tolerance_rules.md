# Zero-Tolerance Rules

These rules are non-negotiable and must be enforced at all times.

1. No Hardcoded Secrets
2. No SQL Injection
3. No XSS Vulnerabilities
4. No Unhandled Errors
5. No Missing Tests (80% coverage minimum)
6. No Undocumented Code
7. No Duplicate Code (DRY)
8. No Uncommitted Changes
9. No Direct DOM Manipulation (Frontend)
10. No Bypassing Validation
