# دليل البدء السريع لنظام Gaara ERP

## مقدمة

مرحباً بك في نظام Gaara ERP المتكامل مع نظام الذكاء الاصطناعي الزراعي. هذا الدليل سيساعدك على البدء في استخدام النظام بسرعة وسهولة.

## متطلبات النظام

قبل البدء في استخدام نظام Gaara ERP، تأكد من توفر المتطلبات التالية:

- نظام تشغيل: Linux، Windows، أو macOS
- Python 3.8 أو أحدث
- PostgreSQL 12 أو أحدث
- متصفح ويب حديث (Chrome، Firefox، Safari، أو Edge)
- اتصال بالإنترنت (للوصول إلى خدمات الذكاء الاصطناعي)

## تثبيت النظام

### 1. تحميل النظام

```bash
git clone https://github.com/your-organization/gaara-erp.git
cd gaara-erp
```

### 2. إنشاء بيئة افتراضية

```bash
python3 -m venv venv
source venv/bin/activate  # على Linux/macOS
# أو
venv\Scripts\activate  # على Windows
```

### 3. تثبيت المتطلبات

```bash
pip install -r requirements.txt
```

### 4. إعداد قاعدة البيانات

```bash
# إنشاء قاعدة بيانات PostgreSQL
createdb gaara_erp

# تشغيل سكربت إعداد قاعدة البيانات
python scripts/setup_database.py
```

### 5. إعداد ملف البيئة

قم بنسخ ملف `.env.example` إلى `.env` وتعديل القيم حسب إعدادات بيئتك:

```bash
cp .env.example .env
```

ثم قم بتعديل الملف باستخدام محرر النصوص المفضل لديك:

```
# إعدادات قاعدة البيانات
DB_HOST=localhost
DB_PORT=5432
DB_NAME=gaara_erp
DB_USER=postgres
DB_PASSWORD=your_password

# إعدادات الخادم
SERVER_HOST=0.0.0.0
SERVER_PORT=2050
DEBUG=true

# إعدادات الذكاء الاصطناعي
AI_API_URL=http://localhost:8000/api
AI_API_KEY=your_api_key
```

## تشغيل النظام

### 1. تشغيل الخادم

يمكنك تشغيل الخادم باستخدام سكربت التشغيل المرفق:

```bash
chmod +x run_server.sh
./run_server.sh
```

أو يمكنك تشغيله مباشرة باستخدام Python:

```bash
python src/main.py --port 2050 --host 0.0.0.0
```

### 2. الوصول إلى النظام

بعد تشغيل الخادم، يمكنك الوصول إلى النظام من خلال المتصفح على العنوان التالي:

```
http://localhost:2050
```

## تسجيل الدخول

عند الوصول إلى النظام لأول مرة، ستظهر صفحة تسجيل الدخول. استخدم بيانات الاعتماد الافتراضية:

- اسم المستخدم: `admin`
- كلمة المرور: `admin123`

**ملاحظة هامة**: يجب تغيير كلمة المرور الافتراضية فور تسجيل الدخول لأول مرة.

## الإعداد الأولي

عند تسجيل الدخول لأول مرة، سيطلب منك النظام إكمال الإعداد الأولي:

1. **إعداد معلومات الشركة**: أدخل اسم الشركة، العنوان، معلومات الاتصال، والشعار.
2. **إعداد الدول والفروع**: أضف الدول والفروع التي تعمل بها الشركة.
3. **إعداد المستخدمين والصلاحيات**: أضف المستخدمين وحدد صلاحياتهم.
4. **إعداد الإعدادات العامة**: حدد العملة الافتراضية، التقويم، وإعدادات أخرى.

## استخدام النظام

### لوحة التحكم الرئيسية

بعد تسجيل الدخول والإعداد الأولي، ستظهر لوحة التحكم الرئيسية التي تعرض:

- ملخص الإحصائيات الرئيسية
- الرسوم البيانية للأداء
- التنبيهات والإشعارات
- المهام المعلقة

### التنقل في النظام

يمكنك التنقل في النظام باستخدام القائمة الجانبية (Sidebar) التي تحتوي على جميع وحدات النظام:

- **المشاتل والمزارع**: إدارة المشاتل، المزارع، المخزون، والعمليات.
- **الذكاء الاصطناعي**: تشخيص الأمراض النباتية وتهجين النباتات.
- **المخزون**: إدارة الأصناف، المستودعات، وحركات المخزون.
- **الحسابات**: إدارة القيود اليومية، دليل الحسابات، والتقارير المالية.
- **الموارد البشرية**: إدارة الموظفين، الأقسام، الحضور، والرواتب.
- **التقارير**: عرض وإنشاء التقارير المختلفة.
- **الإدارة**: إدارة المستخدمين، الصلاحيات، وإعدادات النظام.

## استخدام وحدة الذكاء الاصطناعي

### تشخيص الأمراض النباتية

1. انتقل إلى قائمة "الذكاء الاصطناعي" ثم اختر "تشخيص الأمراض".
2. انقر على زر "إنشاء طلب تشخيص جديد".
3. قم بتحميل صورة النبات المصاب.
4. حدد نوع النبات وأدخل وصفاً للحالة.
5. انقر على زر "إرسال" لبدء عملية التشخيص.
6. انتظر حتى تكتمل عملية التشخيص واعرض النتائج.

### تهجين النباتات

1. انتقل إلى قائمة "الذكاء الاصطناعي" ثم اختر "تهجين النباتات".
2. انقر على زر "إنشاء طلب تهجين جديد".
3. حدد أصناف الآباء من القائمة.
4. حدد أهداف التهجين والقيود.
5. انقر على زر "إرسال" لبدء عملية التهجين.
6. انتظر حتى تكتمل عملية التهجين واعرض النتائج.

## إدارة المشاتل والمزارع

### إدارة المشاتل

1. انتقل إلى قائمة "المشاتل" ثم اختر "لوحة تحكم المشاتل".
2. استعرض إحصائيات المشاتل والرسوم البيانية.
3. لإضافة مشتل جديد، انقر على زر "إضافة مشتل جديد".
4. أدخل معلومات المشتل (الاسم، الموقع، السعة، المدير).
5. لإدارة مخزون المشتل، انتقل إلى "مخزون المشاتل".
6. لإدارة عمليات المشتل، انتقل إلى "عمليات المشاتل".

### إدارة المزارع

1. انتقل إلى قائمة "المزارع" ثم اختر "لوحة تحكم المزارع".
2. استعرض إحصائيات المزارع والرسوم البيانية.
3. لإضافة مزرعة جديدة، انقر على زر "إضافة مزرعة جديدة".
4. أدخل معلومات المزرعة (الاسم، الموقع، المساحة، نوع التربة، المدير).
5. لإدارة قطع الأراضي، انتقل إلى "قطع الأراضي".
6. لإدارة عمليات المزرعة، انتقل إلى "عمليات المزارع".

## التقارير

1. انتقل إلى قائمة "التقارير".
2. اختر نوع التقرير المطلوب (مالي، مخزون، مشاتل، مزارع، موارد بشرية).
3. حدد معايير التقرير (الفترة الزمنية، الفرع، القسم).
4. انقر على زر "إنشاء التقرير".
5. يمكنك تصدير التقرير بصيغ مختلفة (PDF، Excel، CSV).

## الدعم والمساعدة

إذا واجهتك أي مشكلة أو كان لديك أي استفسار، يمكنك:

1. الاطلاع على دليل المستخدم الشامل في قسم "المساعدة".
2. التواصل مع فريق الدعم الفني عبر البريد الإلكتروني: support@gaara-erp.com
3. الاتصال بالرقم: +123456789
