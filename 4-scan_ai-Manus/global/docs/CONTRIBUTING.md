# Contributing

We welcome contributions to this project! Please follow these guidelines to ensure a smooth process.

---

## How to Contribute

1.  **Fork the repository.**
2.  **Create a new branch** for your feature or bug fix:
    ```bash
    git checkout -b feat/my-new-feature
    ```
3.  **Make your changes.**
4.  **Write tests** for your changes.
5.  **Ensure all tests pass:**
    ```bash
    npm test
    ```
6.  **Commit your changes** with a descriptive commit message (following the [Conventional Commits](https://www.conventionalcommits.org/) specification).
7.  **Push your branch** to your fork:
    ```bash
    git push origin feat/my-new-feature
    ```
8.  **Create a pull request** to the `main` branch of the original repository.

## Code of Conduct

Please note that this project is released with a Contributor Code of Conduct. By participating in this project you agree to abide by its terms.
