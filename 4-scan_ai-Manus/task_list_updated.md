# قائمة المهام الرئيسية والفرعية المحدثة

## 1. نظام مقارنة الأصناف النباتية
- [ ] إنشاء نموذج بيانات لمواصفات الأصناف
- [ ] تطوير واجهة إدخال بيانات الأصناف الجديدة
- [ ] إنشاء نظام لتحديد أماكن التجارب والمساحات المزروعة
- [ ] تطوير نظام مقارنة متعدد المعايير (لون الثمرة، حجم الثمرة، شكل الثمرة، التحمل، المقاومة، الإنتاجية)
- [ ] إضافة إمكانية تحميل الصور للأصناف
- [ ] تطوير واجهة إدارة معايير المقارنة للمسؤولين
- [ ] إنشاء نظام تقييم المستخدم للأصناف

## 2. نظام إدارة المشاتل
- [ ] إنشاء نموذج بيانات للمشاتل والشتلات
- [ ] تطوير نظام إدارة طلبات الشتلات
- [ ] إنشاء نظام تتبع دورة حياة الشتلة (البذر، النمو، الاكتمال)
- [ ] تطوير نظام حساب الطاقة الاستيعابية للمشتل
- [ ] إنشاء نظام حجز الشتلات وتحديد مواعيد التسليم
- [ ] تطوير نظام تقييم أداء الشتلات
- [ ] إضافة دعم للمواسم المختلفة (صيف/شتاء)

## 3. وكيل الذكاء الاصطناعي
- [ ] تطوير واجهة المحادثة مع وكيل الذكاء الاصطناعي
- [ ] إنشاء نظام تحميل وتنزيل الملفات
- [ ] تصميم شخصية الوكيل (فرعون صغير)
- [ ] تطوير نظام تطور الوكيل مع التعلم (زيادة عمر الفرعون)
- [ ] ربط الوكيل بنظام الصلاحيات
- [ ] إنشاء نسختين من الوكيل (مجاني ومدفوع)
- [ ] تطوير نظام الإجابة على الأسئلة المتخصصة

## 4. نظام التقارير المتقدم
- [ ] تطوير تقارير على مستوى النظام
- [ ] إنشاء تقارير مدخلات المستخدمين
- [ ] تطوير تقارير ملفات التعلم وإجراءات التعلم
- [ ] إنشاء تقارير تحديثات قواعد البيانات
- [ ] تطوير نظام تصدير التقارير بصيغ متعددة
- [ ] إنشاء لوحات معلومات تفاعلية

## 5. تنظيم هيكل المحاصيل
- [ ] إعادة تنظيم المجلدات حسب أنواع المحاصيل (خضروات، فواكه، محاصيل)
- [ ] تطوير نظام تصنيف الشتلات
- [ ] إنشاء قواعد بيانات منفصلة لكل نوع
- [ ] تطوير واجهة تصفح حسب نوع المحصول

## 6. تحسين نظام الصلاحيات
- [ ] تطوير صلاحيات إضافية للمستخدمين
- [ ] إنشاء نظام موافقة الرئيس قبل الدمج
- [ ] تطوير واجهة إدارة الصلاحيات للمسؤول
- [ ] إنشاء نظام تتبع التغييرات حسب المستخدم

## 7. نظام التكوين البيئي
- [ ] إنشاء ملفات .env للبيانات الحساسة
- [ ] تطوير نظام مفاتيح API لكل مستخدم
- [ ] إنشاء نظام إدارة المفاتيح
- [ ] تطوير نظام تأمين البيانات

## 8. إدارة المصادر الموثوقة
- [ ] إنشاء قاعدة بيانات للمصادر الموثوقة
- [ ] تطوير نظام التحقق من المصادر
- [ ] إنشاء ملف للمصادر الموثوقة للنظام
- [ ] تطوير ملف شامل للمصادر للمسؤول
- [ ] إنشاء نظام توثيق المعلومات والتواريخ

## 9. تكامل واختبار النظام
- [ ] دمج جميع المكونات الجديدة
- [ ] إجراء اختبارات شاملة للنظام
- [ ] تحسين الأداء والكفاءة
- [ ] توثيق النظام المحدث
- [ ] إنشاء دليل المستخدم المحدث
