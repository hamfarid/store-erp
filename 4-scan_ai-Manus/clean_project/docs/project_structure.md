```
│   ├── modules/                      # وحدات النظام
│   │   ├── ai_management/            # إدارة الذكاء الاصطناعي
│   │   ├── disease_diagnosis/        # تشخيص الأمراض
│   │   ├── image_search/             # البحث عن الصور
│   │   ├── internal_diagnosis/       # التشخيص الداخلي
│   │   ├── performance_monitoring/   # مراقبة الأداء
│   │   └── plant_disease/            # أمراض النباتات
│   ├── web_interface/                # واجهة الويب
│   │   ├── admin_panel/              # لوحة الإدارة
│   │   └── frontend/                 # الواجهة الأمامية
│   └── services/                     # الخدمات المساعدة
└── docker-compose.yml                # تكوين Docker (إذا وُجد)
```

## الوحدات الرئيسية

### 1. وحدة إدارة الذكاء الاصطناعي (ai_management)
- إدارة نماذج التعلم الآلي
- تدريب وتحديث النماذج
- مراقبة الأداء والدقة

### 2. وحدة تشخيص الأمراض (disease_diagnosis)
- تحليل الصور باستخدام الذكاء الاصطناعي
- تحديد نوع المرض ودرجة الإصابة
- تقديم توصيات العلاج

### 3. وحدة البحث عن الصور (image_search)
- البحث في قواعد البيانات المحلية والخارجية
- جمع وتصنيف الصور المرجعية
- نظام البحث الذاتي الذكي

### 4. وحدة التشخيص الداخلي (internal_diagnosis)
- تشخيص حالة النظام
- مراقبة الأداء والأخطاء
- تقارير الصحة العامة

### 5. وحدة مراقبة الأداء (performance_monitoring)
- مراقبة استخدام الموارد
- تحليل الأداء والاستجابة
- إنذارات وتنبيهات

### 6. وحدة أمراض النباتات (plant_disease)
- قاعدة بيانات الأمراض النباتية
- محلل أنماط الانتباه
- معالج متقدم للأمراض

## واجهة الويب

### لوحة الإدارة (admin_panel)
- إدارة المستخدمين والصلاحيات
- مراقبة النظام والإحصائيات
- إعدادات النظام والتكوين

### الواجهة الأمامية (frontend)
- واجهة المستخدم الرئيسية
- رفع وتحليل الصور
- عرض النتائج والتوصيات

## الملفات الأساسية

### main.py
الملف الرئيسي لتشغيل النظام، يحتوي على:
- إعداد الخادم الرئيسي
- تكوين المسارات والواجهات
- تهيئة الخدمات

### database.py
إدارة قاعدة البيانات، يشمل:
- اتصالات قاعدة البيانات
- نماذج البيانات
- عمليات CRUD

### ai_service.py
خدمة الذكاء الاصطناعي، تتضمن:
- تحميل وإدارة النماذج
- معالجة الطلبات
- واجهات برمجة التطبيقات

### disease_diagnosis_service.py
خدمة تشخيص الأمراض، تحتوي على:
- تحليل الصور
- تشخيص الأمراض
- إنتاج التقارير
