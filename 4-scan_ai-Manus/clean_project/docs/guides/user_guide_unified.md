## مقدمة

نظام Gaara Scan AI هو منصة متكاملة للتحليل والمعالجة باستخدام الذكاء الاصطناعي، مع تركيز خاص على التطبيقات الزراعية. يوفر النظام مجموعة من الوحدات المتخصصة التي تعمل معًا لتقديم حلول شاملة للمستخدمين.

## متطلبات النظام

- نظام تشغيل: Windows 10/11 أو Ubuntu 20.04+ أو macOS 11+
- المعالج: 4 أنوية على الأقل
- الذاكرة: 8 جيجابايت RAM على الأقل
- مساحة التخزين: 50 جيجابايت على الأقل
- متصفح حديث: Chrome أو Firefox أو Edge
- اتصال إنترنت مستقر

## التثبيت والإعداد

### التثبيت باستخدام Docker (موصى به)

1. قم بتثبيت Docker و Docker Compose:
   - [تثبيت Docker على Windows](https://docs.docker.com/desktop/install/windows-install/)
   - [تثبيت Docker على macOS](https://docs.docker.com/desktop/install/mac-install/)
   - [تثبيت Docker على Ubuntu](https://docs.docker.com/engine/install/ubuntu/)

2. قم بتنزيل مشروع Gaara Scan AI:
   ```bash
   git clone https://github.com/your-organization/gaara-scan-ai.git
   cd gaara-scan-ai
   ```

3. قم بإنشاء ملف البيئة:
   ```bash
   cp .env-example .env
   ```

4. قم بتعديل ملف `.env` حسب احتياجاتك.

### التثبيت بدون Docker

1. قم بتثبيت Python 3.9+ والمتطلبات الأخرى:
   ```bash
   # على Ubuntu
   sudo apt update
   sudo apt install python3 python3-pip python3-venv

   # على Windows
   # قم بتنزيل وتثبيت Python من الموقع الرسمي
   ```

2. قم بإنشاء بيئة افتراضية وتفعيلها:
   ```bash
   python3 -m venv venv
   
   # على Linux/macOS
   source venv/bin/activate
   
   # على Windows
   venv\Scripts\activate
   ```

3. قم بتثبيت المتطلبات:
   ```bash
   pip install -r requirements.txt
   ```

## بدء التشغيل

### باستخدام Docker

استخدم السكريبت الموحد لبدء التشغيل:

- **على Windows (PowerShell)**:
  ```powershell
  .\scripts\start_unified.ps1 -Action start -Environment development
  ```

- **على Linux/macOS**:
  ```bash
  ./scripts/start_unified.sh start development
  ```

### بدون Docker

- **على Windows (PowerShell)**:
  ```powershell
  .\scripts\start_unified.ps1 -Action start -WithoutDocker
  ```

- **على Linux/macOS**:
  ```bash
  ./scripts/start_unified.sh start --without-docker
  ```

بعد بدء التشغيل، افتح المتصفح وانتقل إلى `http://localhost:8000` للوصول إلى واجهة النظام.

## الواجهة الرئيسية

بعد تسجيل الدخول، ستظهر لوحة التحكم الرئيسية التي تحتوي على:

1. **شريط التنقل العلوي**: يحتوي على اسم المستخدم، الإشعارات، وزر تسجيل الخروج.
2. **القائمة الجانبية**: تحتوي على جميع الوحدات المتاحة للمستخدم حسب صلاحياته.
3. **منطقة العرض الرئيسية**: تعرض محتوى الوحدة المحددة.
4. **شريط الحالة السفلي**: يعرض حالة النظام والإصدار الحالي.

## الوحدات الأساسية

### وحدة البحث عن الصور

تتيح هذه الوحدة البحث عن الصور باستخدام تقنيات الذكاء الاصطناعي:

1. انقر على "البحث عن الصور" في القائمة الجانبية.
2. قم بتحميل صورة أو اختر من الصور المتاحة.
3. حدد معايير البحث (التشابه، اللون، النمط، إلخ).
4. انقر على "بحث" للحصول على النتائج.
5. يمكنك تصفية النتائج وتصديرها حسب الحاجة.

### وحدة إدارة الذكاء الاصطناعي

تتيح هذه الوحدة إدارة نماذج الذكاء الاصطناعي المستخدمة في النظام:

1. انقر على "إدارة الذكاء الاصطناعي" في القائمة الجانبية.
2. استعرض النماذج المتاحة وحالتها.
3. قم بتحديث النماذج أو تدريبها حسب الحاجة.
4. راقب أداء النماذج وضبط الإعدادات.

### الوحدات الزراعية

تشمل الوحدات الزراعية مجموعة من الأدوات المتخصصة:

#### تشخيص الأمراض النباتية

1. انقر على "تشخيص الأمراض" في القائمة الجانبية.
2. قم بتحميل صورة للنبات المصاب.
3. انقر على "تشخيص" للحصول على النتائج.
4. استعرض التشخيص والتوصيات المقترحة.

#### إدارة المزارع

1. انقر على "إدارة المزارع" في القائمة الجانبية.
2. استعرض المزارع المسجلة أو أضف مزرعة جديدة.
3. أدخل بيانات المزرعة (الموقع، المساحة، المحاصيل، إلخ).
4. تتبع حالة المزرعة والمحاصيل.

#### تجارب الأصناف

1. انقر على "تجارب الأصناف" في القائمة الجانبية.
2. استعرض التجارب الحالية أو أنشئ تجربة جديدة.
3. أدخل بيانات التجربة (الصنف، الظروف، المعايير، إلخ).
4. سجل النتائج وقارن بين الأصناف.

## إدارة المستخدمين والصلاحيات

### إضافة مستخدم جديد (للمسؤولين فقط)

1. انقر على "إدارة المستخدمين" في القائمة الجانبية.
2. انقر على "إضافة مستخدم جديد".
3. أدخل بيانات المستخدم (الاسم، البريد الإلكتروني، كلمة المرور، إلخ).
4. حدد دور المستخدم وصلاحياته.
5. انقر على "حفظ" لإضافة المستخدم.

### تعديل الصلاحيات

1. انقر على "إدارة المستخدمين" في القائمة الجانبية.
2. ابحث عن المستخدم المطلوب.
3. انقر على "تعديل" بجانب اسم المستخدم.
4. قم بتعديل الصلاحيات حسب الحاجة.
5. انقر على "حفظ" لتطبيق التغييرات.

## استكشاف الأخطاء وإصلاحها

### مشكلة: لا يمكن الوصول إلى واجهة النظام

**الحل**:
1. تأكد من أن الخدمات قيد التشغيل:
   ```bash
   docker-compose ps
   ```
2. تحقق من سجلات الخطأ:
   ```bash
   docker-compose logs app
   ```
3. تأكد من أن المنفذ 8000 غير مستخدم بواسطة تطبيق آخر.

### مشكلة: خطأ في تسجيل الدخول

**الحل**:
1. تأكد من صحة بيانات تسجيل الدخول.
2. تحقق من اتصال قاعدة البيانات:
   ```bash
   docker-compose logs db
   ```
3. إعادة تعيين كلمة المرور إذا لزم الأمر (اتصل بمسؤول النظام).

### مشكلة: بطء في أداء النظام

**الحل**:
1. تحقق من موارد النظام (CPU، RAM، مساحة التخزين).
2. قم بتنظيف ذاكرة التخزين المؤقت:
   ```bash
   .\scripts\start_unified.ps1 -Action restart -Clean
   ```
3. تحقق من سجلات الأداء:
   ```bash
   docker stats
   ```
