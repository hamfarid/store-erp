# File Upload Prompt

## Purpose
Implement secure file uploads.

## Instructions
1. Validate file types
2. Limit file sizes
3. Scan for malware
4. Store files securely
5. Generate unique filenames
6. Log uploads to system_log.md

## Security Measures
- Whitelist file types
- Virus scanning
- Separate storage
- Access control

## Output
- File upload endpoint
- Storage configuration
