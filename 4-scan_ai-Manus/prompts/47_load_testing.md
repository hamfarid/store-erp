# Load Testing Prompt

## Purpose
Test application performance under load.

## Instructions
1. Define load test scenarios
2. Set performance benchmarks
3. Run load tests
4. Analyze results
5. Identify bottlenecks
6. Log test results to system_log.md

## Load Test Scenarios
- Normal load
- Peak load
- Stress test
- Endurance test

## Output
- Load test report
- Performance recommendations
