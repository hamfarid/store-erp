# مسارات الواجهة الأمامية - Gaara ERP v12

## نظرة عامة
توثيق شامل لجميع مسارات الواجهة الأمامية المقترحة لنظام Gaara ERP v12.

## 1. المسارات الأساسية
- `/` - الصفحة الرئيسية
- `/dashboard` - لوحة التحكم
- `/login` - تسجيل الدخول
- `/logout` - تسجيل الخروج
- `/profile` - الملف الشخصي

## 2. مسارات المحاسبة
- `/accounting` - الصفحة الرئيسية للمحاسبة
- `/accounting/accounts` - شجرة الحسابات
- `/accounting/journal-entries` - القيود اليومية
- `/accounting/reports` - التقارير المالية

## 3. مسارات المخزون
- `/inventory` - الصفحة الرئيسية للمخزون
- `/inventory/products` - كتالوج المنتجات
- `/inventory/stock-moves` - حركات المخزون
- `/inventory/warehouses` - المستودعات

## 4. مسارات المبيعات
- `/sales` - الصفحة الرئيسية للمبيعات
- `/sales/customers` - العملاء
- `/sales/orders` - أوامر البيع
- `/sales/invoices` - الفواتير

## 5. مسارات المشتريات
- `/purchasing` - الصفحة الرئيسية للمشتريات
- `/purchasing/suppliers` - الموردين
- `/purchasing/orders` - أوامر الشراء
- `/purchasing/receipts` - إيصالات الاستلام

**تاريخ التوثيق**: نوفمبر 2025
**إصدار النظام**: Gaara ERP v12 Enhanced Security Edition
