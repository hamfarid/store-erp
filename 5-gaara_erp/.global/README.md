# .global/ - Helper Files and Tools

This directory contains all helper programs, scripts, tools, templates, and documentation for Global Guidelines.

## Structure

```
.global/
├── tools/           # Development and analysis tools
├── scripts/         # Integration and setup scripts  
├── templates/       # Project templates and boilerplates
├── examples/        # Code examples and samples
├── flows/           # Workflow documentation
├── guides/          # Integration and usage guides
├── docs/            # Additional documentation
├── backups/         # Project backups
├── archive/         # Old versions and deprecated files
└── workflows/       # CI/CD and automation workflows
```

## Tools

Professional tools for project analysis, maintenance, and optimization.

## Scripts

Integration scripts for setting up and configuring Global Guidelines in projects.

## Templates

Ready-to-use project templates for various frameworks and use cases.

## Examples

Code samples and complete example projects demonstrating best practices.

## Usage

All tools and scripts are designed to work from the project root directory:

```bash
# Run tools
python3 .global/tools/project_analyzer.py

# Run scripts
bash .global/scripts/integrate.sh

# Use templates
cp -r .global/templates/web_page my_project/
```

## Note

This directory is meant to be used as a reference and helper resource. The actual project files (prompts, unified guidelines, setup script) are at the root level.
