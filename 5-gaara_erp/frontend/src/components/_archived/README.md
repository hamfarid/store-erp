# مجلد الملفات المؤرشفة

هذا المجلد يحتوي على الملفات التي تم نقلها من المشروع الرئيسي:

## الملفات المنقولة:

### ملفات مكررة:
- `CustomersAdvanced.jsx` - نسخة قديمة من إدارة العملاء
- `LotManagement.jsx` - نسخة قديمة من إدارة اللوطات  
- `SuppliersAdvanced.jsx` - نسخة قديمة من إدارة الموردين
- `SimpleLogin.jsx` - نسخة قديمة من تسجيل الدخول

### ملفات اختبار:
- `ApiTest.jsx` - اختبار APIs
- `ButtonTest.jsx` - اختبار الأزرار
- `LoginTest.jsx` - اختبار تسجيل الدخول
- `StyleTest.jsx` - اختبار الأنماط
- `IntegrationTest.jsx` - اختبار التكامل

### ملفات غير مستخدمة:
- `Admin.jsx` - واجهة إدارة قديمة
- `ComingSoon.jsx` - صفحة قريباً
- `Login.jsx` - تسجيل دخول قديم

## سبب النقل:
هذه الملفات تم نقلها لتنظيف المشروع والتركيز على الملفات النشطة فقط.

## ملاحظة:
يمكن حذف هذا المجلد بالكامل إذا لم تعد هناك حاجة لهذه الملفات.
